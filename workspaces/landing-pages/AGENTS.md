# Landing Pages Workspace

**Agent:** Landing Page Generator
**Model tier:** Intermediate + Elite for copy

## Inputs
- Product brief (from Planner or user)
- Brand assets

## Outputs
- Next.js + Tailwind page code
- Pushed to `webspace/landing-pages/` for Railway deploy

## Tools available
- File write
- Git (push to deploy branch)

## Conventions
- Each product gets its own subfolder: `landing-pages/<product-slug>/`
- Uses AgentAI template as base
